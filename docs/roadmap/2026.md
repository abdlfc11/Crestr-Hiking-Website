# 2026 Roadmap

A focused list of medium‑scale features and improvements planned for completion during 2026.  
These items strengthen Crestr’s core experience and lay the groundwork for the major 2027 projects.

## Achieved
- [x] Route Elevation Profiles
- [x] Snap to Paths for Manual Routing
- [x] Elevation Data in Manual Routing
- [x] Tag Data (e.g., trail_visibility, surface) in Pathfinding
- [x] Exporting Routes (e.g., GPX)
- [x] Screenshots and Visuals in Documentation
- [x] Undo/Redo for Manual Routing
- [x] Importing routes (e.g, .GPX, .GeoJSON, .FIT)

## Planned
- [ ] GPX Elevation Correction (SRTM/DEM fallback)
- [ ] Improved Elevation Chart (smoothing + stats)
- [ ] POI Markers (summits, tarns, shelters)
- [ ] Route Sharing (link or QR)
- [ ] Offline Tile Caching (Web)
- [ ] Performance Improvements for A* (heuristic tuning)
- [ ] Pre‑computed Regional Graphs (UK National Parks)
- [ ] Basic Public API (route generation endpoint)
- [ ] UI/UX Polish Pass (map interactions, animations)
