# 2027 Roadmap

A forward‑looking overview of Crestr’s major planned features and architectural upgrades for 2027.  
These items represent large‑scale, multi‑month projects scheduled for development between January 2027 and December 2027.

- [ ] Public API v2 (Advanced Routing Options)
- [ ] Multi-Profile Routing (Hiking / Trail Running / Scrambling)

## Planned

- [ ] R-Trees for Spatial Indexing
- [ ] Tile-Based Processing + Pathfinding
- [ ] Difficulty + Greenery Filters for Pathfinding
- [ ] Bi-Directional A* (Full Transition)
- [ ] Expand Automatic Routing Beyond the Lake District
- [ ] Offline Routing Mode (Tiles + Precomputed Graphs)
- [ ] Advanced Elevation Pipeline (Smoothing + Multi-DEM Support)
