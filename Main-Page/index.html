About Crestr

Crestr is an open source hiking app for UK walkers. It lets you plan, save, load and download routes through a clean and focused interface without unnecessary clutter.

Crestr uses open map data and a custom routing engine to build routes between your chosen points. It prioritises safer and greener paths by default, and a feature is planned to let users adjust a path filter to their preferences.
